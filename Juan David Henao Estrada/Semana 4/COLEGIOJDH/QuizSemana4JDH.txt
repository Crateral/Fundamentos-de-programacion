Quiz Base de datos

1. Que es y cuales son las sentencias DML
SELECT: consulta registros de la base de datos es decir los lista.

INSERT: incrustar o insertar datos en una tabla especifica fr ls base de datos.

UPDATE: modifica o mas bien actualiza los registros de los campos en una tabla. 

DELETE: elimina registros en una tabla de una base de datos.  

2. Que es y cuales son las sentencias DDL
Lenguaje Definicion de Datos (DDL) se trabaja mas sobre las tablas y no su contenido
como son:
CREATE: crea nuevas tablas y acompa√±ada de CREATE DATABSE nombre base de datos, crea base 
de datos.

ALTER: Alterar o mas bien modificar las tablas ya sea agregando campos o modificando la 
definicion de los campos. 

DROP: Elimina tablas como tambien indices.

TRUNCATE: Borra el contenido o mas bien los registros completos de una tabla.
 
RENAME: renombra objetos.

3. Describa las 3 primeras formas normales
	1... definir llaves primarias, no tener registros repetidos.
	2... como requisito debe estar en la primera forma normal, se crean 
	tablas para los valores duplicados, se implementa las claves foraneas.
	3... requisito debe estar en la segunda forma normal, se eliminan campos que no dependen de la clave.
