2. Nombre, explique y de un ejemplo de 3 tipos de herencia
	- Herencia única= donde las subclases heredan las características de solo una superclase
		A --> B
	- Herencia multinivel= una clase derivada heredará una clase base y, además, la clase derivada 
	  también actuará como la clase base de otra clase.
		A --> B --> C
	- Herencia Jerárquica= una clase sirve como una superclase (clase base) para más de una subclase.
 
	        C <-- A --> B
		      |
		      v
		      D

4. Cuales son los tipos de datos primitivos?

los tipos de datos primitivos son un conjunto de tipos de datos básicos a partir de los cuales se construyen 
todos los demás tipos de datos
Los tipos de datos primitivos sirven para gestionar los tipos de información más básicos 
siendo un tipo de dato heredado de lenguajes de programacion no orientada a objetos.
Son ocho en total, seis de ellos están destinados a facilitar el trabajo con números enteros (Char, Byte, 
Short, Int, Long),decimales (Float, Double) y los Booleanos(Boolean (verdadero o falso)).

                this.calcularAltura = calcularAltura;

                this.nacer = nacer;
