1. Defina con sus propias palabras que es:
- Una clase: en programación orientada a obojetos es un objeto, el cual posee atributos y métodos 
por ejemplo en una clase puedo agrupar ciertos objetos o cosas por caracteristicas, tamaños, etc.


-Atributo: son las caracteristicas que tiene cada objeto que está dentro de una clase. 

-Metodo: un metodo hace parte de las instrucciones o funcionalidades que posee un objeto perteneciente a una clase.


2. Nombre, explique y de un ejemplo de 3 tipos de herencia
la herencia es una forma de reutilizar u optimizar codigo, existen varios tipos de herencias:

Herencia única: aqui las clases hijas heredan solamente de la clase padre.

herencia multinivel: una clase hija hereda de una clase padre pero ademas esta le hereda caracteristicas a otra
sub clase.

herencia jerarquica: una clase sirve como clase padre para mas de una clase hija.
