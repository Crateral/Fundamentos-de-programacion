1. Que es y cuales son las sentencias DML

las sentencias DML son el lenguaje de manipulacion de datos 
que son aquellas que sirven para insertar, borrar,modificar y consultar
la informacion de una base de datos.
SELECT:consultar datos
INSERT:insertar valores en una base de datos
UPDATE: modificar los valores 
DELETE: elimina las fila de una tabla 

2. Que es y cuales son las sentencias DDL

las sentencias DDl son el lenguaje de definicion datos
son aquellas utilizadas para la creacion de una base de datos y su 
estructura y sus objetos.

CREATE:crear objetos en la base de datos
ALTER:modificar o alterar objetos en la base de datos
DROP:borrar objetos de la base de datos
TRUNCATE:elimina los registros de la tabla
RENAME:cambiarbel nombre de una tabla o un indice

1FN :se trata de identificar los grupos de repeticion, la tabla deve de tener una
